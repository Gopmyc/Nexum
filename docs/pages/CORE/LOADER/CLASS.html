    Overview


    The core Loader module of Nexum.
    This module is responsible for initializing the Loader system, reading and parsing configuration files (YAML), managing libraries and sub-loaders, and providing runtime APIs for instantiation, update, draw cycles, and runtime instance management.
    It isolates libraries in controlled environments, manages client/server/shared state, ensures proper load order, and integrates sub-loaders.

    LOADER:Initialize


    Initializes the Loader with given configuration path and libraries table.
    Validates input parameters, loads YAML configurations, sets up conditional logging via MsgC, creates the loader instance, and initializes sub-loaders.

    Parameters

    • string : Path to the main configuration file (usually 'configuration/nexum.lua')
    • table : Table of available libraries to load

    Returns

    • table : Initialized loader instance ready to manage libraries, runtime, and sub-loaders
    local tLibs = {}
local loader = LOADER:Initialize("configuration/nexum.lua", tLibs)

    LOADER:LoadConfiguration


    Recursively loads all YAML configuration files in a directory.
    Parses YAML files into Lua tables, merges them into a hierarchical configuration table.
    Supports server/client filtering when loading specific folders.

    Parameters

    • string : Path to the folder containing YAML configuration files
    • table : Table of libraries
    • table : Optional table to merge configuration into
    • boolean : Optional flag indicating root call (default true)

    Returns

    • table : Complete merged configuration table
    local tConfig = LOADER:LoadConfiguration("configuration/loader", tLibraries)

    LOADER:CreateLoaderInstance


    Creates a loader instance from configuration.
    Initializes LIBRARIES.BUFFER, sandboxed library environments, ENV_BUILDER, RUNTIME, and RESSOURCES.
    Applies external library references and prepares loader for sub-loaders.

    Parameters

    • table : Loader configuration table
    • table : Table of available libraries

    Returns

    • table : Loader instance
    local tLoader = LOADER:CreateLoaderInstance(tConfig, tLibraries)

    LOADER:InitializeSubloaders


    Initializes sub-loaders according to configuration.
    Loads module groups in order specified by LOAD_PRIORITY, registering resources per sub-loader.

    Returns

    • table : Table of initialized sub-loader instances
    loader:InitializeSubloaders()

    LOADER:GetSubLoaderBase


    Returns the base sub-loader object initialized by InitializeSubloaders.

    Returns

    • table : Sub-loader base instance
    local subBase = loader:GetSubLoaderBase()

    LOADER:LoadSubLoader


    Loads a specific sub-loader from a path and content table.
    Handles client/server loading rules and shared flags.
    Wraps loaded sub-loader with __ID and __PARENT metadata.

    Parameters

    • string : Path to sub-loader Lua file
    • table : Content table of the sub-loader
    • boolean : Whether the sub-loader is shared
    • string : Optional ID for the sub-loader
    • table : Optional environment profile

    Returns

    • table : Loaded sub-loader and its content
    local tSub = loader:LoadSubLoader("path/to/subloader.lua", contentTable, true, "SUB_01")

    LOADER:GetConfig


    Returns the full loader configuration table.

    Returns

    • table : Loader configuration
    local tConfig = loader:GetConfig()

    LOADER:GetLibrariesBase


    Provides a table representing a set of libraries with buffered instances.
    Supports server/client/shared prefixes and auto-loading via ENV_LOADER.
    Registers parent environment access for sub-libraries.

    Parameters

    • string : Base path to the library folder
    • table : Optional parent environment table

    Returns

    • table : Libraries base object with Load method and BUFFER
    local tLibBase = loader:GetLibrariesBase("srcs/core/loader/libraries", parentEnv)

    LOADER:GetLibrary


    Retrieves a loaded library by name from LIBRARIES.BUFFER.
    Shows an error message if the library is not found.

    Parameters

    • string : Name of the library

    Returns

    • table : Library instance
    local runtimeLib = loader:GetLibrary("RUNTIME")

    LOADER:PrintLibraries


    Prints all currently loaded libraries to console with colored MsgC formatting.
    Handles empty or uninitialized LIBRARIES.BUFFER tables gracefully.

    loader:PrintLibraries()

    LOADER:Instanciate


    Creates a runtime instance of a module/entity.
    Fetches sub-loader group and runtime configuration, sets instance ID, and delegates instantiation to RUNTIME library.

    Parameters

    • string : File name of the module
    • string : Instance name
    • vararg : Optional constructor arguments

    Returns

    • table : Runtime instance
    local entity = loader:Instanciate("player.lua", "Player1", param1, param2)

    LOADER:Update


    Executes update cycle for all runtime instances.
    Delegates to RUNTIME library.

    Parameters

    • vararg : Arguments passed to runtime update
    loader:Update(dt)

    LOADER:Draw


    Executes draw cycle for all runtime instances.
    Delegates to RUNTIME library.

    Parameters

    • vararg : Arguments passed to runtime draw
    loader:Draw()

    LOADER:Quit


    Destroys all runtime instances managed by RUNTIME library.
    Should be called when exiting to release resources.

    loader:Quit()
