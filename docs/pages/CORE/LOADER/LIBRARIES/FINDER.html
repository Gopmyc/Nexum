    Overview


    The FINDER library provides utility functions for locating Lua files within directories.
    It supports recursive scanning and returns full paths for all Lua files found in a given folder structure.

    FINDER:GetLuaFiles


    Recursively retrieves all Lua files within a specified folder path.
    Builds a shared table of file paths, including files from nested directories.

    Parameters

    • string : Folder path to search
    • table : Optional shared table to collect file paths

    Returns

    • table : Table containing full paths of all Lua files found
    local tLuaFiles = FINDER:GetLuaFiles("modules/my_module")
