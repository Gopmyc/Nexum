    Overview


    The RUNTIME library manages runtime instances of scripts, including their update and draw pipelines.
    It handles instantiation, registration, destruction, and safe execution with error handling.

    RUNTIME:SetRuntimeConfig


    Sets the runtime configuration table defining UPDATE and DRAW stages.

    Parameters

    • table : Runtime configuration table
    RUNTIME:SetRuntimeConfig({
	UPDATE = { PRE = 0, MAIN = 1, POST = 2 },
	DRAW   = { BACKGROUND = 0, FOREGROUND = 1 }
})

    RUNTIME:Instantiate


    Creates a new instance of a script from RESSOURCES.
    Registers the instance to the update and draw pipelines according to runtime configuration.

    Parameters

    • string : File name of the script in RESSOURCES
    • table : Runtime configuration for this file {ID, UPDATE, DRAW}
    • table : Optional arguments passed to Initialize method

    Returns

    • table : Instantiated object
    local inst = RUNTIME:Instantiate("PLAYER_CONTROLLER", tConfig, {player})

    RUNTIME:RegisterInstance


    Registers an already instantiated object into the runtime update and draw pipelines.
    Orders are determined by stage and optional order values in runtime config.

    Parameters

    • table : Instance object
    • table : Update stage configuration {STAGE, ORDER}
    • table : Draw stage configuration {STAGE, ORDER}
    RUNTIME:RegisterInstance(inst, {STAGE="MAIN"}, {STAGE="FOREGROUND"})

    RUNTIME:DestroyInstance


    Safely destroys a runtime instance.
    If called during update/draw, destruction is deferred until after the pipeline finishes.

    Parameters

    • table : Instance to destroy
    RUNTIME:DestroyInstance(inst)

    RUNTIME:DestroyAllInstances


    Destroys all registered instances immediately.

    RUNTIME:DestroyAllInstances()

    RUNTIME:Update


    Executes the UPDATE pipeline for all registered instances in order of priority.
    Errors in Update calls are caught and logged, and faulty instances are destroyed.

    RUNTIME:Update(deltaTime)

    RUNTIME:Draw


    Executes the DRAW pipeline for all registered instances in order of priority.
    Errors in Draw calls are caught and logged, and faulty instances are destroyed.

    RUNTIME:Draw()

    RUNTIME:GetInstanceByID


    Retrieves an instance by its group ID and optional instance ID.
    If no ID is given, returns the first instance in the group.

    Parameters

    • string : Group ID (typically the file name)
    • string : Instance ID

    Returns

    • table : Instance if found
    local inst = RUNTIME:GetInstanceByID("PLAYER_CONTROLLER", "MAIN")
