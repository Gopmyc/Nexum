    Overview


    The SUBLOADER_BASE library manages the creation, validation, and initialization of sub-loaders within the main loader.
    It ensures file integrity, proper environment setup, and provides accessors for sub-loaders and file groups.

    SUBLOADER_BASE:Initialize


    Creates a new SUBLOADER_BASE instance and initializes attributes like CONFIG, BASE_PATH, and LOADER reference.

    Parameters

    • table : Configuration table for the loader
    • string : Base path for resolving sub-loader scripts
    • table : Reference to the parent loader

    Returns

    • table : Initialized SUBLOADER_BASE object
    local subLoader = SUBLOADER_BASE:Initialize(tConfig, "addons/myaddon/", Loader)

    SUBLOADER_BASE:InitializeGroup


    Initializes a sub-loader group by loading its configuration, validating its content,
    and calling the sub-loader's Initialize method.

    Parameters

    • string : Group name to initialize

    Returns

    • table : Sub-loader object
    • table : Initialized content of the sub-loader
    local sub, content = subLoader:InitializeGroup("WEAPONS")

    SUBLOADER_BASE:CheckFileStructureIntegrity


    Validates a file against the loader's RULES_FILES to ensure required fields and types are correct.

    Parameters

    • number : File ID or index
    • table : File data table to validate

    Returns

    • boolean : True if valid, otherwise outputs an error
    subLoader:CheckFileStructureIntegrity(1, tFile)

    SUBLOADER_BASE:SetAttribute


    Sets a key-value attribute in the sub-loader instance.
    Useful for storing references or metadata.

    Parameters

    • string : Attribute key
    • any : Attribute value
    subLoader:SetAttribute("DEBUG_MODE", true)

    SUBLOADER_BASE:GetAttribute


    Retrieves a previously set attribute by key.

    Parameters

    • string : Attribute key

    Returns

    • any : Value of the attribute
    local debug = subLoader:GetAttribute("DEBUG_MODE")

    SUBLOADER_BASE:GetSubLoaders


    Returns a table containing all initialized sub-loaders.

    Returns

    • table : Table of sub-loaders
    local allSub = subLoader:GetSubLoaders()

    SUBLOADER_BASE:GetSubLoader


    Retrieves a specific sub-loader by group name.

    Parameters

    • string : Group name

    Returns

    • table : Sub-loader object
    local weaponsLoader = subLoader:GetSubLoader("WEAPONS")

    SUBLOADER_BASE:GetGroupByFileName


    Finds which group a specific file belongs to within all sub-loaders.

    Parameters

    • string : File name (case-insensitive)

    Returns

    • string : Group name
    • number : File ID within the group
    • table : File data table
    local group, id, file = subLoader:GetGroupByFileName("AK47")
