📘 Nexum

    Overview


    The COMMAND library of the Relay module in Nexum.
    It handles parsing and executing user commands for the relay system.

    LIBRARY:RunCommand


    Parses a command line string and executes the corresponding command library.
    Supports `/command args` syntax.
    Handles invalid commands and errors gracefully with console messages.

    Parameters

    • table : Relay instance (tRelay)
    • string : Command line to execute, starting with `/`
    local tRelay = CORE:Initialize()
LIBRARY:RunCommand(tRelay, "/download networking")
