📘 Nexum

    Overview


    The MANIFEST library of the Relay module in Nexum.
    It manages module manifests, including registration, validation, link retrieval, and installation status tracking.

    LIBRARY:SetManifest


    Sets the global manifest table for the relay system.
    Prevents overwriting if a manifest is already registered.

    Parameters

    • table : Manifest table to register
    LIBRARY:SetManifest(tManifest)
