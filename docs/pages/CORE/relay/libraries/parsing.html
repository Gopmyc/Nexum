📘 Nexum

    Overview


    The PARSING library of the Relay module in Nexum.
    It provides utilities for parsing module manifests from raw content strings into structured tables.

    LIBRARY:ParseManifest


    Parses a raw manifest string into a Lua table.
    Each non-empty line becomes an entry in the returned table.

    Parameters

    • string : Raw manifest content as a string

    Returns

    • table : Parsed manifest table, each line as an entry
    local rawContent = [[
file1.lua
file2.lua
config/settings.lua
]]
local tManifest = LIBRARY:ParseManifest(rawContent)
