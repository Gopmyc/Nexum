    Color


    Creates a color table from red, green, and blue values.
    The resulting table includes:
    - Numeric indices [1], [2], [3] for R, G, B components
    - [4] for the alpha channel, always 255
    - __hex field representing the color as a hexadecimal string (e.g., "#FF00FF")
