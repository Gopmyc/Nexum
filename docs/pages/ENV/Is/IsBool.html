    IsBool


    Checks whether a given value is of type boolean.
    Returns true if the value is a boolean (`true` or `false`), otherwise returns false.

    Parameters

    • any : The value to check.

    Returns

    • boolean : `true` if `v` is a boolean, `false` otherwise.
    print(IsBool(true))   -- Output: true
print(IsBool(false))  -- Output: true
print(IsBool(0))      -- Output: false
print(IsBool("true")) -- Output: false
