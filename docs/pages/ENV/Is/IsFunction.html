    IsFunction


    Checks whether the given value is a function.
    Returns `true` if the value is a Lua function, otherwise `false`.

    Parameters

    • any : The value to check.

    Returns

    • boolean : `true` if `v` is a function, `false` otherwise.
    print(IsFunction(print))       -- Output: true
print(IsFunction(123))         -- Output: false
print(IsFunction(function() end)) -- Output: true
