    IsString


    Checks whether the given value is a string.
    Returns `true` if the value is of type string, otherwise `false`.

    Parameters

    • any : The value to check.

    Returns

    • boolean : `true` if `v` is a string, `false` otherwise.
    print(IsString("Hello"))   -- Output: true
print(IsString(123))       -- Output: false
print(IsString(""))        -- Output: true
