    LoadFileInEnvironment


    Loads a Lua file from disk into a specified environment table and returns it as a callable chunk.
    The function automatically adapts to the current runtime backend:
    - lovr (Lovr engine)
    - love (LÃ–VE engine)
    - glua (Garry's Mod Lua)
    - standard Lua (io + load)

    It reads the file (or 'init.lua' if the path is a directory), compiles it, and binds it to the given environment.
    If any read or compilation error occurs, it prints an error message and returns nil.

    Parameters

    • string : Path to the Lua file to load
    • table : Table to use as the environment for the loaded chunk

    Returns

    • function : Compiled Lua function ready to be executed, or nil if an error occurred
    local env = {}
local fChunk = LoadFileInEnvironment("scripts/init.lua", env)
if fChunk then
    fChunk()
end
