    table.Copy


    Creates a copy of the given table.
    Supports both shallow and deep copying.
    Deep copy will recursively duplicate all nested tables, avoiding infinite loops by tracking already-seen tables.

    Parameters

    • table : The table to copy.
    • boolean : If true, perform a deep copy; otherwise, a shallow copy.
    • table|nil : Internal tracking table for already-seen tables during deep copy (do not provide manually).

    Returns

    • table : The copied table.
    local tOriginal = {a = 1, b = {c = 2}}
local tShallow = table.Copy(tOriginal)             -- Shallow copy
local tDeep    = table.Copy(tOriginal, true)       -- Deep copy
tDeep.b.c = 3                                    -- Does not affect tOriginal.b.c
